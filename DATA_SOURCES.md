## 📝 Как составить запрос

### Структура письма:

```
От: Студент [ФИО], Горный университет им. Екатерины II
Научный руководитель: [ФИО, должность]

Тема: Запрос данных для дипломного проекта

Прошу предоставить данные геотехнического мониторинга
для дипломного проекта "Цифровой двойник участка метро СПб":

1. Временные ряды показаний датчиков (6-12 месяцев)
2. Данные об осадках грунта/зданий
3. Деформации обделки тоннелей
4. Отчеты по мониторингу

Данные будут использованы только в научных целях.
Готов подписать соглашение о конфиденциальности.
```
